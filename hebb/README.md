## What is Hebbian Learning?
Hebbian Learning is a neural learning rule based on the idea that neurons that fire together strengthen their connection.

In practice, weights are updated based on the correlation between input and output:
- If two neurons activate together, the connection becomes stronger
- Otherwise, the connection weakens or remains unchanged

## Implementation
This folder contains a simple Python implementation of the Hebbian Learning rule, demonstrating how weights are updated based on input patterns.

The implementation includes:
- A base algorithm
- Tests with logical gates (AND, OR, NAND, NOR)
